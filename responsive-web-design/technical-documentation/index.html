Introduction

Kotlin is a modern programming language developed by JetBrains and first introduced in 2011.

Designed to improve developer productivity, Kotlin offers simpler, clearer, and more expressive syntax compared to Java.

Kotlin is fully compatible with Java and can run on the Java Virtual Machine (JVM) platform, allowing developers to use existing Java code directly.

Syntax

Let's start with the following code: 

fun main() {
  println("Hello World")
}

The fun keyword is used to declare a function. A function is a block of code designed to perform a particular task. In the example above, it declares the main() function.

The main() function is something you will see in every Kotlin program. This function is used to execute code. Any code inside the main() function's curly brackets {} will be executed.

For example, the println() function is inside the main() function, meaning that this will be executed. The println() function is used to output/print text, and in our example it will output "Hello World".

Output

The println() function is used to output values/print text.
You can add as many println() functions as you want. Note that it will add a new line for each function: 

fun main() {
  println("Hello World 1")
  println("Hello World 2")
  println("Hello World 3")
}

You can also print numbers, and perform mathematical calculations: 

fun main() {
  println(5 + 5) // 10
}

There is also a print() function, which is similar to println(). The only difference is that it does not insert a new line at the end of the output: 

fun main() {
  print("Hello World! ")
  print("Kotlin is Awesome, ")
  print("I am learning Kotlin ")
}

Comments

Comments can be used to explain Kotlin code, and to make it more readable. It can also be used to prevent execution when testing alternative code.

There are two types of comments in Kotlin:

Single-line comments

Single-line comments start with two forward slashes //. Everything after the // will be ignored by the compiler: 

fun main() {
  // This is a single-line comment
  println("Hello World") // This is also a single-line comment
}

Multi-line comments

Multi-line comments start with /* and end with */. Everything between the /* and */ will be ignored by the compiler: 

fun main() {
  /* This is a multi-line comment
  It can be used to write long comments
  that span multiple lines */
  println("Hello World")
}

Variables

Variables are used to store data. In Kotlin, you can declare a variable using the var or val keywords.

var

The var keyword is used to declare a mutable variable (a variable that can be changed): 

fun main() {
  var name = "John Doe" // Mutable variable
  println(name) // Output: John Doe
  name = "Jane Doe" // Change the value of the variable
  println(name) // Output: Jane Doe
}

val

The val keyword is used to declare an immutable variable (a variable that cannot be changed): 

fun main() {
  val name = "John Doe" // Immutable variable
  println(name) // Output: John Doe
  // name = "Jane Doe" // This will cause an error, since the variable is immutable
}

Note that you can also declare a variable without assigning a value to it. In this case, you need to specify the type of the variable: 

fun main() {
  var name: String // Declare a variable without assigning a value
  name = "John Doe" // Assign a value to the variable
  println(name) // Output: John Doe
}

You can also declare a variable with a specific type, and assign a value to it: 

fun main() {
  var name: String = "John Doe" // Declare a variable with a specific type and assign a value to it
  println(name) // Output: John Doe
}

In Kotlin, you can also declare a variable using the typealias keyword. This allows you to create an alias for a type, which can make your code more readable: 

typealias Name = String // Create an alias for the String type
fun main() {
  var name: Name = "John Doe" // Use the alias to declare a variable
  println(name) // Output: John Doe
}

In Kotlin, you can also declare a variable using the lateinit keyword. This allows you to declare a variable without assigning a value to it, and then assign a value to it later: 

lateinit var name: String // Declare a variable without assigning a value
fun main() {
  name = "John Doe" // Assign a value to the variable
  println(name) // Output: John Doe
}

Note that you can only use the lateinit keyword with mutable variables (var), and not with immutable variables (val).

Data Types

In Kotlin, the type of a variable is decided by its value: 

var x = 100 // x is an Int
var y = 100.5 // y is a Double
var z = "Hello World" // z is a String

However, it is possible to specify the type if you want: 

var x: Int = 100 // x is an Int
var y: Double = 100.5 // y is a Double
var z: String = "Hello World" // z is a String

Data types are divided into different groups:

  • Numbers
  • Character
  • Boolean
  • String
  • Array
  • List
  • Set
  • Map

Numbers

Numbers are used to store numeric values. In Kotlin, there are two types of numbers: integers and floating-point numbers.

  • Int: 32-bit signed integer
  • Long: 64-bit signed integer
  • Short: 16-bit signed integer
  • Byte: 8-bit signed integer
  • Float: 32-bit floating-point number
  • Double: 64-bit floating-point number

Integer Types

Integer types are used to store whole numbers (numbers without decimal points).

Int

The Int data type can store whole numbers from -2147483648 to 2147483647: 

var x: Int = 100 // x is an Int

Long

The Long data type can store whole numbers from -9223372036854775808 to 9223372036854775807: 

var x: Long = 100L // x is a Long

Short

The Short data type can store whole numbers from -32768 to 32767: 

var x: Short = 100 // x is a Short

Byte

The Byte data type can store whole numbers from -128 to 127: 

var x: Byte = 100 // x is a Byte

Floating-point Types

Floating-point types are used to store numbers with decimal points.

Float

The Float data type can store numbers with decimal points from -3.40282347E+38 to 3.40282347E+38: 

var x: Float = 100.5F // x is a Float

Double

The Double data type can store numbers with decimal points from -1.7976931348623157E+308 to 1.7976931348623157E+308: 

var x: Double = 100.5 // x is a Double

Character

The Char data type is used to store a single character. It can store any character from the Unicode character set: 

var x: Char = 'A' // x is a Char

Boolean

The Boolean data type is used to store true or false values: 

var x: Boolean = true // x is a Boolean

String

The String data type is used to store a sequence of characters. It can store any text, including letters, numbers, and symbols: 

var x: String = "Hello World" // x is a String

Array

The Array data type is used to store a collection of values of the same type. It can store any type of data, including numbers, characters, and strings: 

var x: Array = arrayOf(1, 2, 3, 4, 5) // x is an Array of Int

List

The List data type is used to store a collection of values of the same type. It can store any type of data, including numbers, characters, and strings: 

var x: List = listOf(1, 2, 3, 4, 5) // x is a List of Int

Set

The Set data type is used to store a collection of unique values of the same type. It can store any type of data, including numbers, characters, and strings: 

var x: Set = setOf(1, 2, 3, 4, 5) // x is a Set of Int

Map

The Map data type is used to store a collection of key-value pairs. It can store any type of data, including numbers, characters, and strings: 

var x: Map = mapOf("John" to 1, "Jane" to 2, "Joe" to 3) // x is a Map of String to Int

Operators

Operators are used to perform operations on variables and values.
The value is called an operand, while the operation (to be performed between the two operands) is defined by an operator.
In the example below, the numbers 100 and 50 are operands, and the + sign is an operator: 

var x = 100 + 50  // The + sign is an operator

Although the + operator is often used to add together two values, like in the example above, it can also be used to add together a variable and a value, or a variable and a variable: 

var x = 100 + 50  // 150
var y = x + 50    // 200 (150 + 50)
var z = x + y     // 350 (150 + 200)

Kotlin divides the operators into the following groups:

  • Arithmetic Operators
  • Assignment Operators
  • Comparison Operators
  • Logical Operators
  • Bitwise Operators

Arithmetic Operators

Arithmetic operators are used to perform common mathematical operations.

Operator Name Description Example Result
+ Addition Adds two operands var x = 100 + 50 150
- Subtraction Subtracts two operands var x = 100 - 50 50
* Multiplication Multiplies two operands var x = 100 * 50 5000
/ Division Divides two operands var x = 100 / 50 2
% Modulus Returns the remainder of two operands var x = 100 % 50 0
++ Increment Increases the value by 1 var x = 100
++x		 101
-- Decrement Decreases the value by 1 var x = 100
--x		 99

Note that the increment and decrement operators can be used in two ways:

  • Prefix: ++x or --x
  • Postfix: x++ or x--
The prefix operator increases or decreases the value before it is used in the expression, while the postfix operator increases or decreases the value after it is used in the expression. 
var x = 100
println(++x) // Output: 101 (prefix increment)
println(x++) // Output: 101 (postfix increment)
println(x)   // Output: 102 (x is now 102)
println(--x) // Output: 101 (prefix decrement)
println(x--) // Output: 101 (postfix decrement)
println(x)   // Output: 100 (x is now 100)

Increment and decrement operators can also be used with other data types, such as Char, Byte, and Short: 

var x: Char = 'A'
println(++x) // Output: B (prefix increment)
println(x++) // Output: B (postfix increment)
println(x)   // Output: C (x is now C)
println(--x) // Output: B (prefix decrement)
println(x--) // Output: B (postfix decrement)
println(x)   // Output: A (x is now A)

Increment and decrement on Boolean, String, and Array: 

var x: Boolean = true
println(++x) // Output: true (prefix increment)
println(x++) // Output: true (postfix increment)
println(x)   // Output: true (x is still true)
println(--x) // Output: false (prefix decrement)
println(x--) // Output: false (postfix decrement)
println(x)   // Output: false (x is now false)

Increment and decrement on Map, Set, and List: 

var x: Map = mapOf("John" to 1, "Jane" to 2, "Joe" to 3)
println(++x) // Output: {John=1, Jane=2, Joe=3} (prefix increment)
println(x++) // Output: {John=1, Jane=2, Joe=3} (postfix increment)
println(x)   // Output: {John=1, Jane=2, Joe=3} (x is still {John=1, Jane=2, Joe=3})
println(--x) // Output: {John=1, Jane=2, Joe=3} (prefix decrement)
println(x--) // Output: {John=1, Jane=2, Joe=3} (postfix decrement)
println(x)   // Output: {John=1, Jane=2, Joe=3} (x is still {John=1, Jane=2, Joe=3})

If...Else

Kotlin has the following conditionals:

  • Use if to specify a block of code to be executed, if a specified condition is true
  • Use else to specify a block of code to be executed, if the same condition is false
  • Use else if to specify a new condition to test, if the first condition is false
  • Use when to specify many alternative blocks of code to be executed

Kotlin If statement

The if statement is used to specify a block of code to be executed, if a specified condition is true. It is the most common conditional statement in Kotlin. 

var x = 100
if (x > 50) { // If x is greater than 50
  println("x is greater than 50") // Output: x is greater than 50
} // No else statement

Kotlin If with Else statement

The condition is evaluated, and if it is true, the block of code is executed. If it is false, the block of code is skipped. 

var x = 100
if (x > 50) { // If x is greater than 50
  println("x is greater than 50") // Output: x is greater than 50
} else { // If x is not greater than 50
  println("x is not greater than 50") // Output: x is not greater than 50
}

Kotlin If with Else If statement

The else if statement is used to specify a new condition to test, if the first condition is false. It is used to test multiple conditions in a single if statement. 

var x = 100
if (x > 50) { // If x is greater than 50
  println("x is greater than 50") // Output: x is greater than 50
} else if (x < 50) { // If x is less than 50
  println("x is less than 50") // Output: x is less than 50
} else { // If x is equal to 50
  println("x is equal to 50") // Output: x is equal to 50
}

Kotlin When statement

The when statement is a replacement for the switch statement in other programming languages. It is used to specify many alternative blocks of code to be executed. 

var x = 100
when (x) { // When x is equal to 100
  100 -> println("x is equal to 100") // Output: x is equal to 100
  50 -> println("x is equal to 50") // Output: x is equal to 50
  0 -> println("x is equal to 0") // Output: x is equal to 0
}

Loops

Loops can execute a block of code as long as a specified condition is reached.
Loops are handy because they save time, reduce errors, and they make code more readable.

While Loop

The while loop is used to execute a block of code as long as a specified condition is true. The condition is evaluated before the block of code is executed. 

var x = 0
while (x < 5) { // While x is less than 5
  println(x) // Output: 0 1 2 3 4
  x++ // Increment x by 1
}

Do While Loop

The do while loop is used to execute a block of code as long as a specified condition is true. The condition is evaluated after the block of code is executed. 

var x = 0
do { // Do while x is less than 5
  println(x) // Output: 0 1 2 3 4
  x++ // Increment x by 1
} while (x < 5)

For Loop

The for loop is used to execute a block of code a specified number of times. It is used to iterate over a range of values or a collection of values. 

for (x in 0..5) { // For x in the range 0 to 5
  println(x) // Output: 0 1 2 3 4 5
}

The for loop can also be used to iterate over a collection of values, such as an array or a list: 

var x = arrayOf(1, 2, 3, 4, 5)
for (i in x) { // For i in the array x
  println(i) // Output: 1 2 3 4 5
}

The for loop can also be used to iterate over a map, using the entries property: 

var x = mapOf("John" to 1, "Jane" to 2, "Joe" to 3)
for ((key, value) in x) { // For (key, value) in the map x
  println("$key = $value") // Output: John = 1 Jane = 2 Joe = 3
}

Break and Continue

The break statement is used to exit a loop, while the continue statement is used to skip the current iteration of a loop and continue with the next iteration. 

for (x in 0..5) { // For x in the range 0 to 5
  if (x == 3) { // If x is equal to 3
    break // Exit the loop
  }
}
for (x in 0..5) { // For x in the range 0 to 5
  if (x == 3) { // If x is equal to 3
    continue // Skip the current iteration
  }
}
